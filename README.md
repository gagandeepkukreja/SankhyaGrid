# SankhyaGrid (सांख्य)

**Distribution Intelligence Platform for Indian DISCOMs**

Last-mile grid visibility, control, and AI-driven insights across feeders, transformers, homes, EVs, and solar systems.

## Live Demo

👉 [Open Dashboard](https://yourusername.github.io/sankhyagrid/)

## Features

- **GIS View** — Interactive SVG network map with all asset types (substations, feeders, transformers, homes, EV chargers, solar inverters)
- **Real-Time Monitoring** — Feeder & transformer parameters with sparkline trends and data health indicators
- **Alerts & Fault Detection** — Severity-based alerts with AI-powered insights (theft detection, overload prediction, EV load shifting)
- **Predictive Modelling** — Weather alerts, EV load forecast, solar generation forecast, transformer risk prediction
- **DER Management** — EV charging control, solar asset monitoring, energy balance visualization
- **AI Document Extraction** — Upload PDF/JPG/PNG/DOCX for automated OCR & LLM-based asset data extraction

## Tech

- React 18 (standalone, no build step)
- Pure SVG interactive map (no external tile dependencies)
- Simulated real-time data (5s refresh cycle)

This is a single `index.html` file. Just enable GitHub Pages on the `main` branch.
